# API Client UI 操作流程

> 本文档基于 v2 布局设计：三区域架构（侧边栏 + 工作区 + 按需抽屉），命令面板为核心交互入口。
> 详见 02-UI设计.md 和 07b-请求编辑视觉规范.md（请求编辑视觉）、07c-侧边栏与命令面板视觉规范.md（侧边栏/Console/Settings 视觉）

### 3.3 URL 输入交互

**输入行为**:
1. 用户输入 URL
2. 实时解析显示 path/params
3. 变量高亮({{variable}} 显示为蓝色)
4. 自动补全建议

**变量检测**:
```
输入: https://{{base_url}}/api/{{version}}/users
    │
    ▼
┌─────────────────────────────────────────┐

│ https://[{{base_url}}]/api/[{{version}}]/users
│                                           │

│ {{base_url}} = "api.example.com"         │
│ {{version}} = "v1"                       │

```

## 6. 脚本编写流程

### 6.1 Pre-request Script

```
[Scripts] tab → [Pre-request]
+----------------------------------------------------------+
|                                                          |
| // 设置当前时间戳                                        |
| pm.variables.set('timestamp', new Date().toISOString());|
|                                                          |
| // 添加自定义 Header                                     |
| pm.request.headers.add({                                 |
|   key: 'X-Request-ID',                                   |
|   value: pm.variables.get('request_id')                  |
| });                                                       |
|                                                          |
| // 发送额外请求获取 token                                |
| const tokenRequest = {                                   |
|   url: pm.variables.get('auth_url') + '/token',          |
|   method: 'POST',                                        |
|   body: {                                                |
|     mode: 'urlencoded',                                  |
|     urlencoded: [                                        |
|       { key: 'client_id', value: 'xxx' },               |
|       { key: 'client_secret', value: 'yyy' }            |
|     ]                                                    |
|   }                                                      |
| };                                                       |
|                                                          |
| pm.sendRequest(tokenRequest, (err, res) => {            |
|   if (!err) {                                           |
|     const { access_token } = res.json();                |
|     pm.variables.set('token', access_token);            |
|   }                                                      |
| });                                                       |
|                                                          |
+----------------------------------------------------------+
```

### 6.2 Post-response Script

```
[Scripts] tab → [Post-response]
+----------------------------------------------------------+
|                                                          |
| // 解析响应                                              |
| const jsonData = pm.response.json();                     |
|                                                          |
| // 测试: 状态码为 200                                    |
| pm.test('Status code is 200', () => {                    |
|   pm.expect(pm.response.status).to.eql(200);              |
| });                                                       |
|                                                          |
| // 测试: 响应包含 users 数组                             |
| pm.test('Response has users array', () => {             |
|   pm.expect(jsonData).to.have.property('users');        |
|   pm.expect(jsonData.users).to.be.an('array');          |
| });                                                       |
|                                                          |
| // 测试: 每个 user 有 id 和 name                         |
| pm.test('Each user has id and name', () => {            |
|   jsonData.users.forEach(user => {                      |
|     pm.expect(user).to.have.property('id');             |
|     pm.expect(user).to.have.property('name');           |
|   });                                                    |
| });                                                       |
|                                                          |
| // 保存数据供后续使用                                    |
| pm.collectionVariables.set('user_count', jsonData.users.length);|
|                                                          |
| // 保存响应示例                                          |
| if (!pm.response.body.includes('example')) {            |
|   console.log('Warning: Response may be missing examples');|
| }                                                        |
|                                                          |
+----------------------------------------------------------+

Test Results:
[✓] Status code is 200
[✓] Response has users array
[✓] Each user has id and name
```

## 17. 大响应体流式渲染

### 17.1 流式加载交互

```
用户点击 Send → 响应体 > 1MB
    │
    ▼
┌──────────────────────────────────────────────────────────────┐
│ 200 OK   1.2s   2.4MB (loading...)                         │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  [Pretty] [Raw] [Preview]                                   │
│                                                               │
│  ┌──────────────────────────────────────────────────────┐    │
│  │ {                                                    │    │
│  │   "users": [                                        │    │
│  │     { "id": 1, "name": "..." },                     │    │
│  │     { "id": 2, "name": "..." },                     │    │
│  │     ...                                             │    │
│  │     ████████████░░░░░░░░░░  Loading 67%...          │    │
│  │   ]                                                 │    │
│  │ }                                                    │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                               │
│  Showing first 500KB of 2.4MB   [Load More] [Download]     │
└──────────────────────────────────────────────────────────────┘
```

**流式渲染策略**：

| 响应大小 | 策略 |
|---------|------|
| < 100KB | 一次性渲染 |
| 100KB - 1MB | 流式渲染，分块追加 |
| 1MB - 10MB | 先渲染前 500KB，底部显示 "Load More" + 进度条 |
| > 10MB | 不自动渲染，提示用户 "Download as File" 或 "Load Preview" |
