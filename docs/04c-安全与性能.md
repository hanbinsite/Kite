## 6. 安全设计

### 6.1 安全措施

| 措施 | 描述 | 实现方式 |
|------|------|----------|
| 网络请求隔离 | 所有请求通过 Rust 后端 | reqwest (Rust 侧) |
| 脚本沙箱 | 进程级隔离执行 | Rust QuickJS (rquickjs) |
| 敏感数据加密 | Vault 加密存储 | AES-256-GCM (aes-gcm) + Argon2id |
| 密钥管理 | 密钥不离开 Rust | keyring (系统密钥链) |
| HTTPS 支持 | 强制 HTTPS 支持 | reqwest rustls-tls |
| SSL 证书管理 | 自签名 CA 信任 | Rust Certificate API |
| CSP | 内容安全策略 | Tauri CSP header |
| 输入验证 | 验证所有输入 | Zod (前端) + Rust 验证 (后端) |
| 依赖审计 | 前端 + 后端 | npm audit + cargo audit |

### 7.2 OAuth 流程

```typescript
export class OAuth2Service {
  async startAuthCodeFlow(
    config: OAuth2Auth,
  ): Promise<OAuth2TokenResponse> {
    const codeVerifier = generateRandomString(128);
    const codeChallenge = await sha256Base64(codeVerifier);
    const state = generateRandomString(32);

    const authUrl = new URL(config.authUrl!);
    authUrl.searchParams.set('response_type', 'code');
    authUrl.searchParams.set('client_id', config.clientId!);
    authUrl.searchParams.set(
      'redirect_uri',
      'apiclient://oauth/callback',
    );
    authUrl.searchParams.set('scope', config.scope ?? '');
    authUrl.searchParams.set('state', state);
    if (config.pkce) {
      authUrl.searchParams.set('code_challenge', codeChallenge);
      authUrl.searchParams.set(
        'code_challenge_method',
        'S256',
      );
    }

    await openExternal(authUrl.toString());

    return new Promise((resolve, reject) => {
      const timeout = setTimeout(
        () => reject(new Error('OAuth timeout')),
        300000,
      );

      const unlisten = listen(
        'oauth-callback',
        (event) => {
          clearTimeout(timeout);
          unlisten.then((fn) => fn());

          const params = new URLSearchParams(
            event.payload as string,
          );
          if (params.get('state') !== state) {
            reject(new Error('State mismatch'));
            return;
          }

          const code = params.get('code');
          if (!code) {
            reject(new Error('No code in callback'));
            return;
          }

          this.exchangeCodeForToken(code, config, codeVerifier)
            .then(resolve)
            .catch(reject);
        },
      );
    });
  }

  private async exchangeCodeForToken(
    code: string,
    config: OAuth2Auth,
    codeVerifier: string,
  ): Promise<OAuth2TokenResponse> {
    return invoke('exchange_oauth_token', {
      code,
      tokenUrl: config.accessTokenUrl,
      clientId: config.clientId,
      clientSecret: config.clientSecret,
      redirectUri: 'apiclient://oauth/callback',
      codeVerifier,
    });
  }
}
```

### 3.3 脚本引擎实现

```rust
use rquickjs::{Context, Function, Runtime, Undefined};
use std::time::Duration;

pub struct ScriptEngine {
    runtime: Runtime,
}

impl ScriptEngine {
    pub fn new() -> Result<Self, String> {
        let runtime = Runtime::new().map_err(|e| e.to_string())?;
        runtime.set_memory_limit(128 * 1024 * 1024); // 128MB
        runtime.set_max_stack_size(1024 * 1024);     // 1MB
        Ok(Self { runtime })
    }

    pub fn execute(
        &self,
        code: &str,
        context: ScriptContext,
        timeout_ms: u64,
    ) -> Result<ScriptResult, String> {
        let ctx = Context::full(&self.runtime).map_err(|e| e.to_string())?;

        let result = ctx.with(|ctx| {
            let pm = create_pm_object(&ctx, &context)?;
            ctx.globals().set("pm", pm)?;

            let console = create_console_object(&ctx)?;
            ctx.globals().set("console", console)?;

            let result: Result<Undefined, rquickjs::Error> = ctx.eval(code);

            match result {
                Ok(_) => Ok(ScriptResult {
                    success: true,
                    logs: get_captured_logs(),
                    variables: get_modified_variables(),
                    error: None,
                }),
                Err(e) => Ok(ScriptResult {
                    success: false,
                    logs: get_captured_logs(),
                    variables: get_modified_variables(),
                    error: Some(format!("{:?}", e)),
                }),
            }
        });

        result
    }
}

fn create_pm_object<'js>(
    ctx: &rquickjs::Ctx<'js>,
    context: &ScriptContext,
) -> Result<rquickjs::Object<'js>, String> {
    let pm = rquickjs::Object::new(ctx.clone()).map_err(|e| e.to_string())?;

    let request = rquickjs::Object::new(ctx.clone()).map_err(|e| e.to_string())?;
    request.set("method", &context.request.method).map_err(|e| e.to_string())?;
    request.set("url", &context.request.url).map_err(|e| e.to_string())?;
    pm.set("request", request).map_err(|e| e.to_string())?;

    if let Some(ref response) = context.response {
        let resp = rquickjs::Object::new(ctx.clone()).map_err(|e| e.to_string())?;
        resp.set("status", response.status).map_err(|e| e.to_string())?;
        resp.set("body", &response.body).map_err(|e| e.to_string())?;
        pm.set("response", resp).map_err(|e| e.to_string())?;
    }

    let variables = rquickjs::Object::new(ctx.clone()).map_err(|e| e.to_string())?;
    let vars_data = context.variables.clone();
    variables
        .set(
            "get",
            Function::new(ctx.clone(), move |name: String| -> Option<String> {
                vars_data.get(&name).cloned()
            }),
        )
        .map_err(|e| e.to_string())?;
    pm.set("variables", variables).map_err(|e| e.to_string())?;

    let test_fn = Function::new(ctx.clone(), |name: String, _fn_val: rquickjs::Function| {
        // 测试执行逻辑
    })
    .map_err(|e| e.to_string())?;
    pm.set("test", test_fn).map_err(|e| e.to_string())?;

    Ok(pm)
}
```

### 3.4 Vault 加密实现

> 使用 `argon2` 进行密钥派生（KDF），`aes-gcm` 进行数据加密。
> 替代原 sodiumoxide 方案：argon2 是 OWASP 推荐的 KDF，aes-gcm 是 NIST 推荐的 AEAD。

```rust
use keyring::Entry;
use aes_gcm::{Aes256Gcm, KeyInit, Nonce};
use aes_gcm::aead::Aead;
use argon2::{Argon2, PasswordHash, PasswordHasher, PasswordVerifier};
use argon2::password_hash::SaltString;
use serde::{Deserialize, Serialize};
use base64::{Engine, engine::general_purpose::STANDARD as BASE64};

#[derive(Serialize, Deserialize)]
struct StoredSecret {
    name: String,
    nonce: String,        // base64 encoded
    ciphertext: String,   // base64 encoded
    salt: String,         // base64 encoded (Argon2 salt)
    updated_at: String,
}

pub struct VaultService {
    key: Option<[u8; 32]>,   // AES-256 key
    salt: Option<SaltString>,
    base_path: std::path::PathBuf,
}

impl VaultService {
    pub fn new(base_path: std::path::PathBuf) -> Self {
        Self {
            key: None,
            salt: None,
            base_path,
        }
    }

    pub fn unlock(&mut self, master_password: &str) -> Result<(), String> {
        // 尝试加载已有 salt（首次解锁则生成新的）
        let salt = self.load_or_create_salt()?;
        self.salt = Some(salt.clone());

        // Argon2id 派生 256-bit AES 密钥
        let argon2 = Argon2::default();
        let password_hash = argon2
            .hash_password(master_password.as_bytes(), &salt)
            .map_err(|e| e.to_string())?;

        // 从 PHC 字符串中提取原始 hash 作为 AES 密钥
        let hash_output = password_hash.hash
            .ok_or("Failed to derive key from Argon2")?;
        let mut key = [0u8; 32];
        let hash_bytes = hash_output.as_bytes();
        key[..hash_bytes.len().min(32)].copy_from_slice(&hash_bytes[..hash_bytes.len().min(32)]);

        self.key = Some(key);

        // 将验证凭据存入系统密钥链（用于后续验证）
        let entry = Entry::new("api-client", "vault-key").map_err(|e| e.to_string())?;
        entry
            .set_password(&BASE64.encode(key))
            .map_err(|e| e.to_string())?;

        Ok(())
    }

    pub fn lock(&mut self) -> Result<(), String> {
        self.key = None;
        self.salt = None;
        let entry = Entry::new("api-client", "vault-key").map_err(|e| e.to_string())?;
        let _ = entry.delete_credential();
        Ok(())
    }

    pub fn is_unlocked(&self) -> bool {
        self.key.is_some()
    }

    pub fn encrypt_secret(&self, name: &str, plaintext: &str) -> Result<(), String> {
        let key_bytes = self.key.as_ref().ok_or("Vault is locked")?;
        let cipher = Aes256Gcm::new_from_slice(key_bytes)
            .map_err(|e| e.to_string())?;

        // 生成随机 96-bit nonce
        let nonce_bytes = Self::random_bytes(12);
        let nonce = Nonce::from_slice(&nonce_bytes);

        let ciphertext = cipher
            .encrypt(nonce, plaintext.as_bytes())
            .map_err(|e| e.to_string())?;

        let stored = StoredSecret {
            name: name.to_string(),
            nonce: BASE64.encode(nonce_bytes),
            ciphertext: BASE64.encode(&ciphertext),
            salt: self.salt.as_ref()
                .map(|s| s.as_str().to_string())
                .unwrap_or_default(),
            updated_at: chrono::Utc::now().to_rfc3339(),
        };

        let vault_path = self.get_secret_path(name);
        let json = serde_json::to_string_pretty(&stored).map_err(|e| e.to_string())?;
        std::fs::write(&vault_path, json).map_err(|e| e.to_string())?;

        Ok(())
    }

    pub fn decrypt_secret(&self, name: &str) -> Result<String, String> {
        let key_bytes = self.key.as_ref().ok_or("Vault is locked")?;
        let cipher = Aes256Gcm::new_from_slice(key_bytes)
            .map_err(|e| e.to_string())?;

        let vault_path = self.get_secret_path(name);
        let json = std::fs::read_to_string(&vault_path).map_err(|e| e.to_string())?;
        let stored: StoredSecret = serde_json::from_str(&json).map_err(|e| e.to_string())?;

        let nonce_bytes = BASE64.decode(&stored.nonce).map_err(|e| e.to_string())?;
        let nonce = Nonce::from_slice(&nonce_bytes);

        let ciphertext_bytes = BASE64.decode(&stored.ciphertext).map_err(|e| e.to_string())?;

        let plaintext = cipher
            .decrypt(nonce, ciphertext_bytes.as_ref())
            .map_err(|_| "Decryption failed — wrong key or corrupted data".to_string())?;

        String::from_utf8(plaintext).map_err(|e| e.to_string())
    }

    fn derive_key(&self, password: &str, salt: &SaltString) -> Result<[u8; 32], String> {
        let argon2 = Argon2::default();
        let password_hash = argon2
            .hash_password(password.as_bytes(), salt)
            .map_err(|e| e.to_string())?;

        let hash_output = password_hash.hash
            .ok_or("Failed to derive key from Argon2")?;
        let mut key = [0u8; 32];
        let hash_bytes = hash_output.as_bytes();
        key[..hash_bytes.len().min(32)].copy_from_slice(&hash_bytes[..hash_bytes.len().min(32)]);
        Ok(key)
    }

    fn load_or_create_salt(&self) -> Result<SaltString, String> {
        let salt_path = self.base_path.join("vault").join(".salt");
        if salt_path.exists() {
            let salt_str = std::fs::read_to_string(&salt_path).map_err(|e| e.to_string())?;
            SaltString::new(&salt_str).map_err(|e| e.to_string())
        } else {
            let salt = SaltString::generate(&mut rand::thread_rng());
            let dir = self.base_path.join("vault");
            std::fs::create_dir_all(&dir).map_err(|e| e.to_string())?;
            std::fs::write(&salt_path, salt.as_str()).map_err(|e| e.to_string())?;
            Ok(salt)
        }
    }

    fn random_bytes(len: usize) -> Vec<u8> {
        use rand::RngCore;
        let mut buf = vec![0u8; len];
        rand::thread_rng().fill_bytes(&mut buf);
        buf
    }

    fn get_secret_path(&self, name: &str) -> std::path::PathBuf {
        self.base_path.join("vault").join(format!("{}.enc.json", name))
    }
}
```

## 12. Web 版架构差异（可选）

> `apps/web/` 为可选的 Web 版本，与桌面版共享前端代码但后端不同。

### 12.1 核心差异

| 能力 | 桌面版 (Tauri) | Web 版 |
|------|---------------|--------|
| 网络请求 | Rust reqwest (无 CORS 限制) | 浏览器 fetch (受 CORS 限制) 或代理服务 |
| 文件系统 | Rust 直接读写 | 无（需后端 API） |
| SQLite | Rust rusqlite | 无（需 IndexedDB 或后端 API） |
| 脚本引擎 | Rust rquickjs | Web Worker + 有限 API |
| 加密/Vault | Rust keyring + argon2 + aes-gcm | Web Crypto API |
| gRPC | Rust tonic (原生 HTTP/2) | grpc-web (需代理) |
| WebSocket | Rust tokio-tungstenite | 浏览器原生 WebSocket API |
| 深链接 | Tauri 自定义协议 | URL hash / query 参数 |
| 自动更新 | Tauri built-in | 需自建 |

### 13.2 插件注册点

```typescript
// packages/core/src/plugin/registry.ts
class PluginRegistry {
  private plugins: Map<string, PluginManifest> = new Map();

  register(plugin: PluginManifest): void {
    this.plugins.set(plugin.id, plugin);
  }

  unregister(id: string): void {
    this.plugins.delete(id);
  }

  async executeHook<K extends keyof PluginHooks>(
    hookName: K,
    ...args: Parameters<NonNullable<PluginHooks[K]>>
  ): Promise<void> {
    for (const plugin of this.plugins.values()) {
      const hook = plugin.hooks[hookName];
      if (hook) {
        await (hook as Function)(...args);
      }
    }
  }
}

export const pluginRegistry = new PluginRegistry();
```
