## CSS 变量命名约定

> 本项目使用 Tailwind CSS 4.x `@theme` 前缀命名。设计文档中使用**设计令牌名**（如 `--brand`），代码实现中使用**实现变量名**（如 `--color-brand`）。详见 08-开发指南.md §2.7。

| 设计令牌 | 实现变量 | Tailwind Utility |
|---------|---------|-----------------|
| `--brand` | `--color-brand` | `text-brand` |
| `--bg-surface` | `--color-bg-surface` | `bg-bg-surface` |
| `--sidebar-width` | `--spacing-sidebar` | `w-sidebar` |
